# داده‌های تحلیل رزم‌افزاری

این پوشه **فقط داده ساختاریافته JSON** نگه می‌دارد. API یا مدل Django جداگانه‌ای ندارد؛ داده از طریق `orchestrator` (در زمان اجرا و هنگام آموزش مدل Ollama) بارگذاری می‌شود.

## فایل‌ها

| فایل | محتوا |
|------|--------|
| `geography/middle_east_geography.json` | زمین، آب‌وهوا، گلوگاه‌ها، ملاحظات نظامی |
| `personnel/middle_east_personnel.json` | نیرو، ذخیره، ساختار شاخه‌ها و رسته‌ها |
| `weapons/middle_east_weapons.json` | تسلیحات و موجودی کشورها |

## جغرافیا

برای هر کشور:

- **terrain** — نوع زمین، ارتفاع، سختی مانور
- **weather** — اقلیم، دما، بارش
- **strategic_features** — بنادر، فرودگاه‌ها، گلوگاه‌ها
- **military_considerations** — مزایا و معایب زمینی

همچنین الگوهای آب‌وهوایی منطقه‌ای و گلوگاه‌های راهبردی در سطح کل دیتاست تعریف شده‌اند.

**کشورها:** سوریه، عراق، ایران، اسرائیل، لبنان، اردن، عربستان، یمن، مصر، ترکیه

## نیروی انسانی

برای هر کشور:

- تعداد **نیروی فعال** و **ذخیره**
- شاخه‌ها: **ground_forces**، **air_force**، **navy**
- در هر شاخه: رسته‌ها، درجات، تعداد و یگان‌های ویژه

**کشورها:** اسرائیل، ایران، ترکیه، مصر، عربستان (پوشش ناقص نسبت به جغرافیا/تسلیحات)

## تسلیحات

دسته‌های اصلی:

| دسته | نمونه |
|------|--------|
| individual_weapons | تفنگ، مسلسل |
| collective_weapons | تانک، جنگنده |
| robotics_ai_weapons | پهپاد |
| conventional_weapons | توپخانه |
| unconventional_weapons | IED |
| chemical/biological/nuclear_weapons | سلاح‌های غیرمتعارف |

هر نوع شامل مشخصات فنی و **توزیع بین کشورها** (تعداد و مدل‌های اصلی) است.

## ویرایش داده

1. فایل JSON مربوطه را ویرایش کنید (ساختار کلیدها را حفظ کنید).
2. مدل را بازسازی کنید:

```bash
python3 manage.py retrain_wargaming_llm --force
```

راه‌اندازی کامل پروژه در [README اصلی](../README.md) توضیح داده شده است.
